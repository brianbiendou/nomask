# Mission 04 — Checklist SEO quotidienne

## Avant publication de chaque article

### Contenu
- [ ] Titre : 50-60 caractères, mot-clé principal en début
- [ ] Meta description (seo_description) : 150-160 caractères, incitative
- [ ] Titre SEO (seo_title) : optimisé pour la recherche, différent du titre éditorial si besoin
- [ ] Mots-clés (seo_keywords) : 3-5 mots-clés séparés par des virgules
- [ ] Excerpt : 1-2 phrases accrocheuses résumant l'article

### Structure de l'article
- [ ] Un seul H1 (le titre)
- [ ] Utiliser des H2 pour structurer les sections
- [ ] Paragraphes courts (3-4 lignes max)
- [ ] Au moins 300 mots (idéalement 800-1500)
- [ ] Inclure le mot-clé principal dans le premier paragraphe
- [ ] Liens internes vers 2-3 articles liés
- [ ] Lien externe vers au moins 1 source fiable

### Image
- [ ] Image principale de qualité (1200×630 px minimum pour OG)
- [ ] Alt text descriptif (image_alt) incluant le mot-clé
- [ ] Crédit photo renseigné (image_credit)

### Catégorie et tags
- [ ] Catégorie correcte sélectionnée
- [ ] Sous-catégorie si applicable
- [ ] 3-5 tags pertinents ajoutés

### URL
- [ ] Slug court, lisible, avec tirets (pas d'accents, pas de mots vides)
- [ ] Exemple : `sommet-g20-afrique-du-sud` et non `le-sommet-du-g20-en-afrique-du-sud-2025`

## Après publication

- [ ] Vérifier l'article en ligne (rendu, images, liens)
- [ ] Partager sur les réseaux sociaux
- [ ] Vérifier l'indexation dans Google Search Console (Inspection d'URL)
