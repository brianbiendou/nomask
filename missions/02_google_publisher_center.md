# Mission 02 — Google Publisher Center

## Objectif
Inscrire NoMask dans Google Publisher Center pour apparaître dans Google Actualités.

## Prérequis
- Google Search Console configuré et vérifié
- Au moins 20-30 articles publiés
- Site en production avec un domaine propre

## Étapes

### 1. Accéder à Publisher Center
- Aller sur https://publishercenter.google.com
- Se connecter avec le compte Google du projet

### 2. Ajouter une publication
- Cliquer sur « Ajouter une publication »
- Nom de la publication : **NoMask**
- URL : `https://nomask.fr`
- Langue principale : Français

### 3. Configurer les sections
Créer des sections correspondant aux catégories :
- International → `https://nomask.fr/international`
- Politique → `https://nomask.fr/politique`
- Société → `https://nomask.fr/societe`
- Économie → `https://nomask.fr/economie`
- Tech & IA → `https://nomask.fr/tech`
- Culture → `https://nomask.fr/culture`
- Science → `https://nomask.fr/science`
- Sport → `https://nomask.fr/sport`
- Style → `https://nomask.fr/style`

### 4. Ajouter le flux RSS
- Section « Flux RSS » : ajouter `https://nomask.fr/rss.xml`

### 5. Logo et branding
- Uploader le logo carré (512×512 px minimum)
- Uploader le logo rectangulaire

### 6. Soumettre pour validation
- Vérifier toutes les informations
- Soumettre la publication
- Délai de validation : quelques jours à quelques semaines

### 7. Après validation
- Vérifier l'apparition dans Google Actualités
- S'assurer que les nouveaux articles apparaissent correctement
